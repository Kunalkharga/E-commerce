🐶 Dog E-Commerce Website

This is a complete e-commerce website built for selling dog products like food, toys, 
accessories, and grooming essentials. The project uses **HTML, CSS, JavaScript, PHP**,
and **MySQL** to provide both frontend and backend functionalities.

---

## 📌 Features

- 🏠 Beautiful, responsive homepage
- 🛍️ Product listing and categories
- 📦 Add to cart and order system
- 👤 User registration and login
- 📊 Admin dashboard for product management
- 💾 MySQL database integration
- 📧 Contact form with backend handling
- 🚀 Fast, mobile-friendly design

---

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Tools:** XAMPP / MAMP / WAMP (for local server)
